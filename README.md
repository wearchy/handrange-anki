# Handrange Paint

ポーカーの 13x13 ハンドレンジ表を色塗りできる静的サイトです。

## 操作

- 左のパレットで色を選択
- 右の表でセルをクリック、または矢印キーで移動
- `Space` で選択中の色を決定
- `1`〜`9` で色を切り替え
- `6` は白の「3人」として塗る
- `E` または `Backspace` でフォールドに戻す
- ドラッグでまとめ塗り
- 初期状態と `Clear` はすべてフォールド
- `答え合わせ` で固定のハンドレンジ表と比較
- `答えを見る` で最初の画像の答えを表示
- `自分の回答に戻る` で答え表示前の練習状態に戻る
- 答え合わせした日時と正解率はブラウザに保存

## GitHub Pages

このフォルダの `index.html`、`styles.css`、`script.js`、`README.md` を GitHub リポジトリに置けば、そのまま GitHub Pages で公開できます。

公開設定は GitHub の `Settings` → `Pages` で、`Deploy from a branch` を選び、公開したいブランチの root を指定してください。
